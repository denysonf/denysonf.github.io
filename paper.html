Behind the Scenes at DigitalOcean

DigitalOcean provides cloud hosting built for developers they are mainly focus on simplicity for rapid deployment. DigitalOcean provides the perfect environment for application deployment without the added complexity of the server configuration, optimization and scale. One other important distinction of DigitalOcean is that they only use solid state disks SSD.[1] 

Hosting Servers

DigitalOcean hosting servers use Hex-Core Intel CPUs with dedicated ECC Ram and RAID SSD storage. They also designed its own hardware for its cloud solution. Each server comes with 2GB of RAM, a 20 GB SSD, and a 1 Gbit/s network interface.[2]

Management Software

DigitalOcean uses a product called ServerPilot which is the ideal alternative to cPanel and Plesk for DigitalOcean. ServerPilot improved speed, security and usability. ServerPilot was built from the ground up as a control panel for DigitalOcean. The benefits of this platform is that it can keep your servers updated and provide monitoring while you manage all of your servers from one place.[3]

Network

DigitalOcean physical nodes are connected via gigabit ethernet to switches, which are uplinked to our aggregators and core routers. All core routers are uplinked via 10-gigE uplinks to providers.

Storage

Storage is one of those areas that DigitalOcean shines, they only use enterprise class solid state drives SSDs on their servers. DigitalOcean is able to provision a droplet in 55 seconds. SSD hard drives also improve performance on the hosting site with faster disk i/o. In addition to using only SSDs DigitalOcean has a flexible API, and the ability to select the nearest data center location.

Groth

As of August 8, 2014, DigitalOcean overtook Rackspace to become the fifth largest public-facing host, growing 850 percent in the last year.[4] DigitalOcean became 4th largest hosting company in under 2 years.[5] If current growth rates persist, DigitalOcean is likely to become the third largest hosting company within the next few months.[6]

Cloud Compute

DigitalOcean hosting servers are KVM Hypervisor based. KVM hypervisor is a virtualization infrastructure that is built in to the Linux kernel. This type of hypervisor works by turning the kernel module on Linux machine into a hypervisor, allowing it to begin hosting virtual servers or as DigitalOcean calls it droplets. Most hypervisors are capable of hosting nearly any guest operating system.[7] If current growth rates persist, DigitalOcean is likely to become the third largest hosting company within the next few months.[8]
